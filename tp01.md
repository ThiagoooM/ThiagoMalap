1. Netflix (recomendación de series y películas)

Problema que resuelve: recomendar contenido que le guste a cada usuario.
Tipo de ML: Supervisado.
Por qué: el sistema aprende con datos de usuarios (qué miran, qué califican) para predecir qué contenido recomendar.
Datos que necesita: - historial de visualización - calificaciones o likes - tiempo que miran cada contenido - género de películas o series.
Los sistemas de recomendación son uno de los usos más comunes del machine learning en plataformas de streaming.

2. Gmail (filtro de spam)

Problema que resuelve: detectar correos basura automáticamente.
Tipo de ML: Supervisado.
Por qué: se entrena con correos etiquetados como “spam” o “no spam”.
Datos que necesita: contenido del correo - remitente - patrones de texto - comportamiento de usuarios marcando spam.
Los servicios de email usan ML para clasificar mensajes y detectar spam

3. Tesla Autopilot

Problema que resuelve: ayudar a conducir automáticamente detectando autos, peatones y señales.
Tipo de ML: Supervisado.
Por qué: se entrena con millones de imágenes y datos de conducción etiquetados.
Datos que necesita: imágenes de cámaras - sensores del auto - datos de tráfico - mapas y rutas.
El sistema usa redes neuronales para interpretar el entorno del vehículo.

4. Spotify (recomendación de música)

Problema que resuelve: sugerir canciones o playlists personalizadas.
Tipo de ML: No supervisado (principalmente).
Por qué: agrupa usuarios y canciones según patrones de escucha similares.
Datos que necesita:canciones escuchadas - tiempo de reproducción - playlists creadas - gustos de usuarios similares.

5. Mercado Libre (Argentina / Latinoamérica)

Problema que resuelve: recomendar productos y detectar fraudes en compras.
Tipo de ML: Supervisado.
Por qué: el sistema aprende de compras pasadas y comportamientos de usuarios para predecir qué mostrar o detectar fraude.
Datos que necesita:historial de compras - búsquedas de productos - clics en publicaciones - datos de vendedores y transacciones.
Mercado Libre utiliza inteligencia artificial y machine learning para mejorar su plataforma de comercio electrónico
